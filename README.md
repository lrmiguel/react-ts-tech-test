# react-ts-tech-test

This is Convex React TypeScript Tech Test.

![Preview of Frontend](public/preview.png)

## Requirements

- npm
- Access to the Internet
- Suitable development environment

## Getting Started

- Run it: `npm start`
- Test it: `npm test`
- Build it: `npm build`
- Run E2E tests: `npm run cypress`
